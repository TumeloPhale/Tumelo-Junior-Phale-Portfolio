# Tumelo Junior Phale

Software Engineer based in South Africa. This is my portfolio — it covers my work, my stack, and how to reach me.

## The site

Built with plain HTML and CSS. No frameworks, no JavaScript, nothing fancy. It loads fast, works on any screen, and does exactly what a portfolio needs to do.

## What you'll find

The Stack section covers languages and tools I actually use. Experience gets into what I've been building professionally. About is a bit of background on me. Contact has everything you need to reach me or download my CV.

## Under the hood

HTML5 and CSS3. Mobile first, semantic markup, accessibility considered throughout. JetBrains Mono for the typography. No build step, no dependencies.
