# Tumelo Junior Phale — Software Engineer Portfolio

A production-grade portfolio showcasing professional experience, technical skills, and engineering capabilities.

## Overview

This portfolio presents my journey as a Graduate Software Engineer at Sage, focusing on backend development, frontend UI improvements, and database management. Built with semantic HTML5, CSS3 design systems, and accessibility-first principles.

## Sections

- **Stack** — Core programming languages and AI tools
- **Experience** — Current role at Sage (Oct 2025–Present) with backend, frontend, and database work
- **About** — Background, skills, and professional philosophy
- **Contact** — Email, GitHub, LinkedIn, and CV download

## Technologies

- **Frontend**: HTML5 (semantic), CSS3 (custom properties, responsive design)
- **Design**: Mobile-first approach, WCAG 2.1 AA accessibility, light mode
- **Typography**: JetBrains Mono (monospace), fluid scaling with clamp()
- **Performance**: Single CSS file, no JavaScript required, optimized font loading

## Design System

The portfolio uses CSS custom properties for maintainability:

- **Colors**: Electric blue primary (#2563eb), grayscale neutrals
- **Typography**: 16px base, 11px–24px scale
- **Spacing**: 4px–68px consistent increments
- **Responsive**: 4 breakpoints (1024px, 768px, 640px, 380px)
